# Appnest Toolkit

The canonical copy of the family build recipe & house style lives in the
Febra repo (the reference implementation):

https://github.com/appnestonline/febra/blob/main/APPNEST_TOOLKIT.md

It covers the visual identity (palette, dark mode, components), the technical
spine (server, client, i18n, tests) and the generic per-app deploy steps that
all four tools - Febra, Tensio, Doza and Mena - follow. This stub exists so
each repo stays self-explanatory without four drifting copies of the same
document.
